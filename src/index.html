Home Page

Welcome to my small application to work with database designed and created to manage and operate a simplificated university. There are very few records in every table for demonstrational purposes and to check if everything is working just fine, you can view the contents of tables by clicking and desired card or using navigation menu attached to the top of the screen. Before starting it's highly recommended to log into system with existing user or create a new one, it currently changes nothing but at least you can do it. And all that with simple and friendly user interface!

Every table implements basic CRUD (Create, Read, Update, Delete) operations and supports ordering the results by clicking at the header of the column you want to sort. In the future there may be more functionality such as different ways of visualizing data and statistics with use of graphs and charts, etc.
